# CLI website

## Overview

This was my 2nd website I ever made (circa: 2023). I created this website to emulate how it feels in a terminal. There are many issues with the website, as I was just learning J.S. at thatpoint in time. It was programmed by hand using only DOM elements. If I was to do this again, I would've built a small terminal emulator, then drawn the elements to a <canvas> rather than to the DOM itself. 

## Why Is This Still Online?

I believe that if I keep my previous websites online. Then whoever looks at what I can do now, can see the hard journey I went through navigating thes older programming methods. This is also so I can look to my past and see how far I've gone aswell.
